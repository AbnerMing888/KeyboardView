# KeyboardView
自定义数字键盘，源文件存放地址

## 文件解读
就两个文件，不值当的打远程Maven了，大家用的话，直接可以复制到项目里即可。

```
KeyboardView：自定义的数字键盘View
attrs：属性文件
```
